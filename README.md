# Cybersecurity Chatbot

## Description
This project is a Cybersecurity Awareness Chatbot developed using C# and WPF in Visual Studio 2022.

The chatbot helps users learn about cybersecurity topics through interactive conversations and dynamic responses.

---

## Features
- GUI developed using WPF
- Keyword recognition
- Random chatbot responses
- Memory and recall system
- Sentiment detection
- Error handling
- Object-Oriented Programming structure

---

## Cybersecurity Topics
The chatbot recognises topics such as:
- Password safety
- Scams
- Privacy
- Phishing
